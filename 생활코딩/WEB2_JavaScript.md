## 15. 비교 연산자와 블리언
- Comparison operator & Boolean
- `document.write(1===1);`라고 치면 true가 출력된다.
- `document.write(1===2);`라고 치면 false가 출력된다.
- `===`는 비교연산자이다. 왼쪽값과 오른쪽값을 비교해서 같으면 true 다르면 false라는 값을 출력한다.
- true와 false를 묶어서 Boolean이라고 한다.
- 즉 데이터타입은 지금까지 number, string, boolean 총 3개이다.
- 이 Boolean을 통해서 인류는 단독반복에서 벗어날 수 있었다.
- `1<2`는 자바스크립트에서 꺽쇠이기 때문에 `1&lt;2` 라고 해야 `1<2`라고 출력된다.

## 21. 반복문
- `document.write(`<li>1</li>`);`라는 코드를 계속 반복해야 한다면
- while을 사용한다. `while()`에서 ()내부가 true이면 계속 반복한다. 
- ()를 한번 돌때마다 확인해서 false이면 반복을 종료한다. 
- 즉 ()은 반복문을 제어하는 제어문이다. 
- 반복문의 밑에 `i = i+1;`을 하고 제어문에 `i<3`을 넣으면 총 3번 반복할 것이다.

- 반복문은 순서대로 배열에 담긴 데이터를 하나씩 꺼내서 자동화된 처리를 할 수 있는 문법이다.

## 36. UI vs API
- UI는 유저인터페이스
- API는 어플리케이션 프로그래밍 인터페이스이다.
- 버튼을 누르면 경고창이 뜰때 이것은 사용자가 쓴다. 이때 유저인터페이스
- 코드를 살펴보면 경고창의 모양이나 길이 이런것은 우리가 만들지 않았다.
- 경고창을 누군가 미리 만들고 약속해놓으면 우리는 그 약속을 믿고 경고창을 사용한다.
- 이렇게 alert처럼 코딩을 할때 쓰는 약속을 API라고한다.
- 프로그래머는 API를 순서대로 실행시키는게 프로그래밍이다.
- 자바스크립트라는 접착제로 API를 조립해서 어플리케이션을 만든다.
